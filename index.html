berry

Berry takes a mapping of routes to functions and constructs a WSGI app. It is a minimal interface to WSGI. Add in your favorite ORM, template engine, and serve it with a WSGI server.

import berry

@berry.get('^hello$')
def hello(req):
  return "Hello, world!"

wsgi_app = berry.app
my_wsgi_server(wsgi_app)
